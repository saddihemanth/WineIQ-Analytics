# Support

Thanks for using or contributing to **WineIQ Analytics**! Here's how to get
help.

## 📖 Documentation

- [README.md](README.md) — overview, installation, usage, architecture
- [CONTRIBUTING.md](CONTRIBUTING.md) — development workflow
- [CHANGELOG.md](CHANGELOG.md) — what changed and when
- [ROADMAP.md](ROADMAP.md) — what's planned next

## 🐛 Found a Bug?

Search [existing issues](https://github.com/saddihemanth/WineIQ-Analytics/issues)
first. If it's new, open a bug report with steps to reproduce, expected vs.
actual behavior, and your environment (OS, Python version).

## 💡 Have a Feature Idea?

Check the [Roadmap](ROADMAP.md), then open an issue tagged `enhancement`
describing the use case.

## 🔐 Security Issues

Please follow the responsible-disclosure process in
[SECURITY.md](SECURITY.md) rather than filing a public issue.

## 💬 General Questions

For setup help, usage questions, or general discussion, open a
[GitHub Discussion](https://github.com/saddihemanth/WineIQ-Analytics/discussions)
(or an issue tagged `question` if Discussions isn't enabled on the repo).
